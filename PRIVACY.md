# Privacy Note - The Food Gap

Last updated: July 22, 2026

## English

The Food Gap is a static public website. The application code does not create accounts, ask for names,
collect form responses, set cookies, use analytics, or store personal information.

The site uses public Statistics Canada data and local app files. The optional basemap loads remote
CARTO map tiles based on OpenStreetMap data. Like any public website, the hosting provider and remote
map-tile provider may process routine technical request information such as IP address, browser, time
of request, and requested file. The Food Gap app itself does not use that information to identify
visitors.

If analytics, feedback forms, mailing lists, or other interactive services are added later, this note
must be updated before public deployment.

## Français

The Food Gap est un site Web public statique. Le code de l'application ne crée pas de comptes, ne
demande pas de noms, ne recueille pas de réponses à des formulaires, n'installe pas de témoins, n'utilise
pas d'outils d'analyse et ne stocke pas de renseignements personnels.

Le site utilise des données publiques de Statistique Canada et des fichiers locaux de l'application. Le
fond de carte facultatif charge des tuiles CARTO à distance, fondées sur des données OpenStreetMap.
Comme pour tout site Web public, l'hébergeur et le fournisseur de tuiles cartographiques peuvent traiter
des renseignements techniques courants liés aux requêtes, comme l'adresse IP, le navigateur, l'heure de
la requête et le fichier demandé. L'application The Food Gap elle-même n'utilise pas ces renseignements
pour identifier les visiteurs.

Si des outils d'analyse, des formulaires de rétroaction, des listes d'envoi ou d'autres services
interactifs sont ajoutés plus tard, cette note devra être mise à jour avant tout déploiement public.
